Project 1: Movie Recommendation System

A machine learning-based project that recommends similar movies based on user input.

Project Overview

This project uses a movie dataset to recommend movies based on features like keywords, genres, popularity, director, and cast. It leverages text processing and cosine similarity to find and recommend similar movies.

Dataset Used

The dataset includes information about movies, such as titles, genres, keywords, popularity, director, and cast details.

Technology Stack

  • Python
  • Pandas
  • NumPy
  • Scikit-learn

Code Snippet


import pandas as pd
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.metrics.pairwise import cosine_similarity

df = pd.read_csv("movie_dataset.csv")
features = ['keywords', 'genres', 'popularity', 'director', 'cast']

# Fill missing values and combine features
for feature in features:
    df[feature] = df[feature].fillna(' ')

df["combine_feature"] = df.apply(lambda row: ' '.join(str(row[f]) for f in features), axis=1)
vectorizer = CountVectorizer()
matrix = vectorizer.fit_transform(df["combine_feature"])
similarity = cosine_similarity(matrix)

def get_recommendations(movie_title):
    index = df[df.title == movie_title].index[0]
    similar_movies = list(enumerate(similarity[index]))
    sorted_movies = sorted(similar_movies, key=lambda x: x[1], reverse=True)
    recommendations = [df.iloc[i[0]].title for i in sorted_movies[1:11]]
    return recommendations

How to Run

To run the project:

  1. Install the necessary libraries: pip install pandas numpy scikit-learn
  2. Download the movie_dataset.csv file and place it in the project folder.
  3. Run the script using python movie_recommendation.py and input a movie title when prompted.

Example Output

If you input "The Godfather", you might get recommendations like:

  • The Godfather: Part II
  • Goodfellas
  • Scarface
  • Pulp Fiction
